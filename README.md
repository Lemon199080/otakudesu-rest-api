# 🌀 Otakudesu Scraper
Sebuah REST API sederhana untuk mengambil data dari [Otakudesu](https://otakudesu.cloud)

- 🔍 Cari anime
- 📘 Detail anime
- 🎞️ Link streaming & download
- 📺 Daftar ongoing & complete
- ⚡ Caching bawaan (pakai `node-cache`)
